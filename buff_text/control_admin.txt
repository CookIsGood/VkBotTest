Вы попали в меню управления админами!
Если вам недоступны кнопки добавить/удалить админа, то скорее всего у вас
права доступа ниже "2".

Для того, что посмотреть список всех админов нажмите на кнопку "Список админов".

Для того, чтобы добавить админа нажмите на кнопку "Добавить админа"
После этого введите id человека, которого хотите сделать админом бота в формате: 111111111
Далее отправьте это сообщение боту, а после нажмите на кнопку "Сохранить добавление"

Для того, чтобы удалить админа для начала нажмите на кнопку "Список админов"
Нажмите на кнопку "Удалить админа"
После этого введите id человека, которого вы хотите удалить из администраторов(убедитесь, что это не ваш id)
Далее отправьте это сообщение боту, а после нажмите на кнопку "Сохранить удаление"

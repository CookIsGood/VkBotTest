Вы попали в меню по рассылке сообщение для проголовавших в опросе!
Для того, чтобы посмотреть текущую рассылку по опросу нажмите на кнопку "Посмотреть рассылку опроса"

Для того, чтобы редактировать сообщение опроса нажмите на кнопку "Создать сообщение"
Введите текст и прикрепите вложение и отправьте его боту.
Далее нажмите на кнопку "Сохранить сообщение"
ВАЖНО!! - Картинка/видео должны быть уже загружены в публичное место(паблик ВК/открытый профиль ВК)

Для того, чтобы контролировать опрос по которому будет присылаться сообщение участникам группы
надо ввести его id
Получение id опроса идет таким образом:
Нажимаете на нужном вопросе "Получить код"(эту вкладку можно увидеть если нажать на три точки справа сверху в опросе)
А потом копируете те цифры, которые показаны во вложение к этому сообщению(цифры могут быть разными)

Для того, чтобы изменить/добавить id опроса нажмите на кнопку "Создать id опроса"
Введите id опроса в формате: 111111111
И отправьте это сообщение боту, а после нажмите на кнопку "Сохранить id опроса"
